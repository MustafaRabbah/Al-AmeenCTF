# طريقة الحل - الرمز الحسابي

## الفكرة
البرنامج يتوقع 6 أرقام. لكل خانة يطبق المعادلة:
- `t = (d * 3 + i * 7) % 10`
ثم يقارن الناتج مع:
- `{9, 0, 6, 4, 3, 2}`

## خطوات الحل
1. استخرج المعادلة والمصفوفة الهدف من دالة `verify`.
2. اعكس المعادلة لكل موضع `i` من 0 إلى 5.
3. بعد العكس ستحصل على الرمز:
   - `314159`
4. أدخل الرمز في البرنامج.
