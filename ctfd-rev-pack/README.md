# Al-Ameen CTF - Reverse Engineering Pack

هذه الحزمة تحتوي على 8 تحديات هندسة عكسية:

- 3 تحديات سهلة
- 2 تحديات متوسطة
- 1 تحدي صعب جدًا
- 2 تحديات Ultra

كل تحدي داخل مجلد مستقل ويحتوي على:

- `description.md` وصف عربي جاهز للمنصة
- `challenge-info.md` معلومات التحدي (الصعوبة، النقاط، طريقة التشغيل)
- `solution.md` طريقة الحل (للمُنظّم)
- `src/challenge.c` كود المصدر
- `bin/` ملف التشغيل بعد البناء
